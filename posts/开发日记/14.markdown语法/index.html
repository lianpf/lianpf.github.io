一、标准基础语法

1. 文案和标题

标题

// 几个#号几级标题。如一个井号是一级标题，四个井号是四级标题
# 一级标题
...
#### 四级标题

加粗、斜体和删除

*text* // 一对*是斜体
**text2** // 两对*是加粗
***text3*** // 多对*号斜粗体

~~11~~ // 两对~~是删除

斜体 加粗 斜体加粗 删除

引用

> 引用 text // 一个>号变引用。即可呈现出灰色底色引用的效果

引用 text

2. 图片和链接

图片

// Alt 和 Title 都不是必须的，可以省略，但建议写上
// Alt的意思是如果图片因为某些原因不能显示，就用定义的图片Alt文字来代替图片
// Title表示鼠标悬停与图片上时出现的文字
![Alt](src “Title”) // 可省略为 ![](src)

超链接

[Tag](href) // 括号里是对应的链接。例：[百度](https://www.baidu.com)

3. 分隔线和列表

分隔线

*** // 三个以上的星号、减号、底线建立一个分割线

列表

// 有序列表
1. Bird
2. McHale

// 无序列表。使用`*`,`+`,`-`中任意一种来表示
- red
+ Green
* Blue

4. 代码块

  • 在3对`号内插代码，表示插入一段高亮代码。
  • 在1对`号内插入代码，表示插入一行高亮代码
@requires_authorization
def somefunc(param1='', param2=0):
    '''A docstring'''
    if param1 > param2: # interesting
        print 'Greater'
    return (param2 - param1 + 1) or None

5. 插入表格

|表示分割线，:表示文案对齐方式

Markdown　Extra表格语法：

| 项目 | 价格 | 数量 
| :--- | ---:| :---: 
| Computer  | 1600 元 |  5   
| Phone     |   12 元 |  12
项目价格数量
Computer1600 元5
Phone12 元12

流程图
