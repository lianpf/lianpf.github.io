此文以 Claude Code 为例，依据 Claude Code 官方文档「快速开始」 整理终端 CLI 的安装与入门流程。

一、Claude Code 安装与使用#

准备：Claude 订阅（Pro、Max、Teams 或 Enterprise）。

1. 安装 和 登录 Claude Code#

1.1 安装#

# 方式1：原生安装（推荐）
# macOS、Linux、WSL
curl -fsSL https://claude.ai/install.sh | bash

# 方式2：Homebrew（macOS）
brew install --cask claude-code
# 官方提供两个 cask：
# 1.claude-code 跟随稳定渠道（通常晚约一周、会跳过存在严重问题的版本）；
# 2.claude-code@latest 跟随最新渠道。
# Homebrew 安装不会自动更新，需定期执行 brew upgrade claude-code 或 brew upgrade claude-code@latest

1.2 配置账户#

首次在交互会话中使用 claude 时会提示登录：

# 首次使用时系统会提示您登录
claude

可以选择在会话中登录， 或者参考 CC-Switch | MiniMax配置 cc-switch：

方式1：/login

/login
# 按照提示使用您的账户登录

方式2：cc-switch（推荐） 参考：cc-switch 添加 MiniMax配置

# 1.配置添加以下内容
API Key: sk-xxx
请求地址：https://xxx-api.xxx.com

# 2.配置模型名称
# 如：将模型名称全部改为 MiniMax-M2.7

# 3.回到首页，点击 “启用” 即可开始使用

# 4.编辑配置文件
# 编辑或新增 .claude.json 文件，MacOS & Linux 为 ~/.claude.json
# 新增 `hasCompletedOnboarding` 参数
{
  "hasCompletedOnboarding": true
}

2. 启动 Claude Code#

在项目目录中启动：

cd /path/to/your/project
claude

进入后可看到界面（会话信息、近期对话、更新等）。

基本命令速查

命令功能示例
claude启动交互模式claude
claude "task"运行一次性任务claude "fix the build error"
claude -p "query"一次性查询后退出claude -p "explain this function"
claude -c在当前目录继续最近对话claude -c
claude -r恢复之前的对话claude -r
claude commit创建 Git 提交claude commit
/clear清除对话历史/clear
/help显示可用命令/help
/resume继续之前的对话/resume
exit 或 Ctrl+C退出 Claude Codeexit

完整参数与命令见官方 CLI 参考。

快捷键：? 查看快捷键；Tab 补全；↑ 历史；输入 / 查看命令与 skills。

二、扩展 Claude Code#

1、claude code + minimax 识别图片#

claude code + minimax 暂不支持识别图片，但可通过 minimax 的 mcp 服务支持

# 1、安装 uvx
mac或linux系统：curl -LsSf https://astral.sh/uv/install.sh  | sh

# 2、配置编辑 ~/.claude.json 文件，新增mcpServers属性。
{
  "mcpServers": {
    "MiniMax": {
      "command": "uvx",
      "args": ["minimax-coding-plan-mcp", "-y"],
      "env": {
        "MINIMAX_API_KEY": "api key",
        "MINIMAX_API_HOST": "https://xxx-api.xxx.com "
      }
    }
  }
}

# 3、用法: claude code 支持 ctrl+v 粘贴图片，用 minimax 的 mcp 服务识别分析图片。
